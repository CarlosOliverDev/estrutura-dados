# Estruturas de Dados e Algoritmos em Java 

Este repositório contém implementações práticas de estruturas de dados clássicas, algoritmos de ordenação e busca, desenvolvidos como parte dos meus estudos fundamentais em Java.

##  Estrutura do Projeto

O projeto está dividido em pacotes para separar a lógica de negócio da execução:

* **`entities`**: Contém a implementação principal das estruturas e algoritmos (o "motor").
* **`application`**: Classes contendo o método `main` para testar e visualizar o funcionamento das implementações.

---

##  Conteúdo do Repositório

### 1. Algoritmos de Ordenação (Sorting)
Implementações dos principais algoritmos de ordenação para análise de complexidade e performance:
* **Bubble Sort**
* **Insertion Sort**
* **Selection Sort**
* **Merge Sort**
* **Quick Sort**
* **TimSort**

### 2. Estruturas de Dados
Implementações customizadas para entender o gerenciamento de memória e ponteiros:
* **Listas:** Lista Vetorizada e Lista Encadeada.
* **Filas:** Fila Circular, Fila Vetorizada e Fila Infinita.
* **Pilhas:** Implementação baseada em vetores.

### 3. Algoritmos de Busca
* **Pesquisa Binária:** Implementação eficiente para vetores ordenados.
